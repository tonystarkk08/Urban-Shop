## Run Locally

Clone the project

```bash
  git clone git@github.com:tonystarkk08/Urban-Shop.git
```

Go to the project directory

```bash
  cd Urban-Shop
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
